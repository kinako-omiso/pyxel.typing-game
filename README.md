# pyxel.typing-game

この度は遊んでいただきありがとうございます。
zipフォルダを解凍し、typing-game.exeファイルをダブルクリックするだけでゲームが始まります。


■ 操作方法\n
・難易度選択：マウスでクリック
・ゲーム開始：スペースキー
・文字入力　：キーボードでタイピング
・決定（判定）：エンターキーを押すことで正誤判定を行います。
・終了　　　：画面右上のドアのアイコンをクリック、またはEscキー
・画面左上のホームアイコンをクリックすることでメニュー画面に戻れます。

■ 起動時に警告が出た場合（Windows）\n
「WindowsによってPCが保護されました」という青い画面が出ることがあります。
これは「作者が不明な自作ソフト」であるために表示されるものです。
以下の手順で実行できます：
1. 「詳細情報」をクリック
2. 「実行」ボタンをクリック

■ 制作\n
このゲームはゲームエンジンpyxelを使用して制作されています。
