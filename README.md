# pyxel.typing-game

この度は遊んでいただきありがとうございます。<br>
zipフォルダを解凍し、typing-game.exeファイルをダブルクリックするだけでゲームが始まります。


■ 操作方法<br>
・難易度選択：マウスでクリック<br>
・ゲーム開始：スペースキー<br>
・文字入力　：キーボードでタイピング<br>
・決定（判定）：エンターキーを押すことで正誤判定を行います。<br>
・終了　　　：画面右上のドアのアイコンをクリック、またはEscキー<br>
・画面左上のホームアイコンをクリックすることでメニュー画面に戻れます。<br>

■ 起動時に警告が出た場合（Windows）<br>
「WindowsによってPCが保護されました」という青い画面が出ることがあります。<br>
これは「作者が不明な自作ソフト」であるために表示されるものです。<br>
以下の手順で実行できます：<br>
1. 「詳細情報」をクリック<br>
2. 「実行」ボタンをクリック<br>
<br>
